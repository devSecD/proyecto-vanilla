# Proyecto Vanilla

Este es un proyecto de ejemplo con HTML, CSS y Javascript puro (sin framework), creado como ejercicio final del curso de Git.

## Tecnologias usadas

- HTML5
- CSS3
- Javascript

## Estructura del proyecto

```
proyecto-vanilla/
├── index.html
├── assets/
│   ├── css/
│   │   └── style.css
│   └── js/
│       └── script.js
└── README.md
```
